# Race Classification using YOLOv4-tiny on AMB82-mini

This repository contains a lightweight object detection and classification model designed to distinguish between three racial categories: **Asian, Black, and White**. 

The project leverages the **YOLOv4-tiny** architecture, optimized specifically for edge computing on the **Realtek AMB82-mini**, a low-cost AIoT development board. This demonstrates the feasibility of running real-time, AI-driven computer vision tasks on highly constrained, affordable edge devices.

* **Three-Class Detection:** Trained to recognize and classify three categories: `Asian`, `Black`, and `White`.
* **Cost-Effective Deployment:** Built for the Realtek AMB82-mini, proving that advanced computer vision can be deployed on budget-friendly development boards.
